# TripleTen_projects
Projects I worked on while attending the Data Analysis TripleTen Bootcamp.
| Project Number | Title | Description 
| :-----------: | ----------- | ----------- |
| 1 | Fresh Beats Listener Trends and Strategic Recommendations Report | This report analyzes user listening trends across genres and tiers to uncover opportunities for catalogue expansion, increased engagement, and stronger free-to-paid conversion.|
| 2 | NYC Air BnB Rental Popularity & Earnings Overview | This project analyzes NYC vacation rental data to identify which neighborhoods and property types attract the most demand. The findings highlight a few key areas and unit sizes that consistently perform well. These insights help hosts and investors focus on the options most likely to generate strong occupancy and revenue. |
| 3 | Exploring Startup Trends with SQL | This project analyzes the educational backgrounds of employees from failed startups and calculates the average education level to uncover meaningful patterns. The findings provide clients with data-driven insight to better understand the role education may or may not play in startup success. |
| 4 | Data Visualization with Tableau | This project examines the Superstore dataset to uncover the key drivers behind profit, loss, and product returns. Through targeted analysis, it identifies which product combinations are most profitable, where advertising efforts should be focused, and which items or customers contribute to high return rates. These insights support smarter merchandising, marketing, and operational decisions to help the business prevent financial decline.
| 5 | Seasonal Geographic Patterns in U.S. Product Returns Data Story | This project explores seasonal and geographic return patterns in Superstore data from 2018 to 2021. Return rates peak in August and December, with Utah showing consistently high returns and Idaho performing more steadily. These insights support smarter inventory planning, targeted marketing, and stronger return-management strategies.
